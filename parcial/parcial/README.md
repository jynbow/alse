# Parcial ALSE - Contar Puntos en Cuadrante

## Descripción
Programa en C++ que cuenta cuántos puntos de un arreglo se encuentran en un cuadrante específico del plano cartesiano.

## Función implementada
`contarPuntosEnCuadrante(Point puntos[], int n, int cuadrante)`

## Cuadrantes
- Cuadrante 1: x > 0, y > 0
- Cuadrante 2: x < 0, y > 0
- Cuadrante 3: x < 0, y < 0
- Cuadrante 4: x > 0, y < 0

## Uso
1. Compilar: `g++ main.cpp -o main`
2. Ejecutar: `./main`
3. Ingresar número de puntos, coordenadas y cuadrante
